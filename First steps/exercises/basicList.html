Listas ordenada e não ordenada

A principio temos duas formas de fazer uma lista no HTML, uma ordenada que é representada pela tag "ol" e as não ordenadas usando a tag "ul". Vale ressaltar que geralmente a desordenada é mais comum para fazer menus, usando uma tag de link e css pra estilizar. Lembrando que todo o compolente da lista é dentro da tag "li"

Lista ordenada:

  1. Banana
  2. Morango
  3. Uva

Lista desordenada:

  • Casa
  • Moradia
  • Morador

Listas de definição são usadas em assuntos onde há um termo a em sequência a sua definição, estilo perguntas e respostas:

HTML
Linguagem de marcação de texto
CSS
Usado para formatação de documentos
